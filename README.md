# OMISS Award Hunter — Downloads

A Windows desktop app for hunting **OMISS** operating awards, by **N6WWW**.

While you're checked into an OMISS net in NetLogger, a live heads-up display ranks which
checked-in stations to work to advance your awards. A companion offline dashboard renders
~110 awards — now with **certificate-level tracking** (each band/tier enhancement counted
the way OMISS issues Cert #s) and an **OMISS sync** that reads your officially-issued awards
by your OM number and shows exactly which certificates you still need to **apply for**.

## Download & install

1. Open the **[latest release](../../releases/latest)** and download
   `OMISS_Award_Hunter_Setup_v<version>.exe`.
2. Run it. It installs **per-user (no admin)** to `%LOCALAPPDATA%\Programs\OMISS Award Hunter`
   and adds a Start-menu shortcut. Uninstall any time from Settings → Apps.

### "Windows protected your PC" warning
The installer is **not code-signed**, so Windows SmartScreen may warn about an unknown
publisher. Click **More info → Run anyway** — this is expected for an independent app.

## Requirements
Windows 10/11. The live check-in features need [NetLogger](https://www.netlogger.org/).
